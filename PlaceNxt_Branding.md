## 2. Logo Concept

### Primary Logo (Wordmark + Icon)

```
  ╭─────╮
  │ ▶ N │  PlaceNxt
  ╰─────╯
```

### Logo Design Specification

| Element | Detail |
|---|---|
| **Icon** | A **rounded square** (like an app icon) containing a stylized **"N"** with a **forward arrow (▶)** embedded in the first leg of the N |
| **Symbolism** | The arrow-in-N represents "next step / moving forward". The rounded square signals "platform / app / digital product" |
| **Wordmark** | "Place" in medium weight + "Nxt" in bold weight (emphasizes the "Nxt" differentiator) |
| **Style** | Geometric, clean, minimal. No gradients in the logo itself (gradients in UI, not logo) |

### Logo Variations

| Variant | Use Case |
|---|---|
| **Full logo** (Icon + Wordmark) | Website header, pitch deck, documents |
| **Icon only** (Rounded square with N▶) | Favicon, mobile app icon, social media avatar |
| **Wordmark only** ("PlaceNxt") | Email signatures, partnerships, small spaces |
| **Monochrome white** | Dark backgrounds |
| **Monochrome dark** | Light backgrounds, print |

### Logo Clear Space

- Minimum padding = height of the icon on all sides
- Never place the logo on busy backgrounds without a solid backing

### Logo Don'ts

- Never stretch or distort
- Never change the font
- Never separate "Place" and "Nxt" onto different lines
- Never add drop shadows or 3D effects to the logo itself
- Never use the old "Medhiva" brain icon with the PlaceNxt name

| Element | Hex | Usage |
|---|---|---|

| Surface 2 | `#F1F5F9` | Elevated elements |

| Text Tertiary | `#94A3B8` | Placeholders |

### Color Rationale

| Decision | Reasoning |
|---|---|
| **Indigo over Purple** | Indigo (`#6366F1`) feels more professional/trustworthy than bright purple. Closer to what LinkedIn, Stripe, and Notion use. Students need to trust the platform with career decisions. |
| **Emerald for success** | Green = "placed successfully" / "test passed" / "badge earned". The most intuitive positive color. |
| **Keeping dark mode default** | Existing dark-mode glassmorphic design is already strong. Students use platforms at night. Dark mode reduces eye strain during long sessions. |
| **Reduced pink** | Pink as primary feels too playful for a career platform. Kept only for premium/special accents. |

## 4. Typography

### Font System

| Role | Font | Weight | Fallback |
|---|---|---|---|
| **Headings** | **Inter** | 600 (Semibold), 700 (Bold) | system-ui, sans-serif |
| **Body** | **Inter** | 400 (Regular), 500 (Medium) | system-ui, sans-serif |
| **Code/Data** | **JetBrains Mono** | 400 | monospace |

### Why Inter over Outfit?

| Factor | Outfit (Previous) | Inter (New) |
|---|---|---|
| Readability at small sizes | Good | Excellent |
| Professional perception | Stylish/trendy | Clean/trustworthy |
| Industry standard | Rare in career tools | Used by GitHub, Figma, Linear |
| Variable font support | Limited | Full variable font |
| Character set | Good | Excellent (multi-language) |
| Dashboard readability | Adequate | Superior (designed for UI) |

### Type Scale

| Level | Size | Weight | Line Height | Usage |
|---|---|---|---|---|
| Display | 48-72px | 700 | 1.1 | Landing page hero |
| H1 | 36-48px | 700 | 1.2 | Page titles |
| H2 | 28-36px | 600 | 1.25 | Section headings |
| H3 | 22-28px | 600 | 1.3 | Card titles |
| H4 | 18-22px | 600 | 1.35 | Sub-section headings |
| Body Large | 18px | 400 | 1.6 | Feature descriptions |
| Body | 16px | 400 | 1.6 | Default body text |
| Body Small | 14px | 400 | 1.5 | Helper text, captions |
| Caption | 12px | 500 | 1.4 | Labels, badges, timestamps |

| Glassmorphism cards | **KEEP** | Distinctive and on-trend for 2026 |

| Smooth animations | **KEEP** | fade-in, slide-up, scale-in |

| Primary color | Purple `#8b5cf6` | Indigo `#6366F1` |
